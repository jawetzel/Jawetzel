# SEO Checklist

Topics to be aware of when bringing an inherited project up to par.

## On-page metadata

- [ ] Title tags
- [ ] Meta descriptions
- [ ] Canonical URLs
- [ ] OpenGraph tags
- [ ] Twitter card tags
- [ ] Structured data (JSON-LD)

## Page semantics

- [ ] H1 / heading hierarchy
- [ ] Image alt text
- [ ] Internal anchor text

## Site structure

- [ ] robots.txt
- [ ] sitemap.xml
- [ ] URL depth (3-click reachability)
- [ ] Trailing-slash convention
- [ ] 301 redirects for migrated URLs
- [ ] Redirect chains

## Crawlability

- [ ] Server-side vs client-side rendering
- [ ] Anti-scraper layer's search-engine bypass
- [ ] 404 status correctness (soft-404 risk)
